Power Manager Driver
Power Manager

This module covers the device-specific clock_manager functionality implemented for S32K1xx and S32MTV SOC.

Hardware background

System mode controller (SMC) is passing the system into and out of all low-power Stop and Run modes. Controls the power, clocks and memories of the system to achieve the power consumption and functionality of that mode.

Driver consideration

Power mode entry and sleep-on-exit-value are provided at initialization time through the power manager user configuration structure. With platform is S32K14x, the available power mode entries are the following ones: HSRUN, RUN, VLPR, STOP1, STOP2 and VLPS. With platform is S32MTV,S32K11x, the available power mode entries are the following ones: RUN, VLPR, STOP1, STOP2 and VLPS.

This is an example of configuration:

&pwrMan1_InitConfig0
